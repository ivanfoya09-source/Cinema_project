- Перегляд сеансів

# ⚙ Встановлення

## 1. Клонувати проєкт

```bash
git clone <repo>
```

---

## 2. Створити віртуальне середовище

```bash
python -m venv .venv
```

---

## 3. Активувати

Windows

```bash
.venv\Scripts\activate
```

Linux/macOS

```bash
source .venv/bin/activate
```

---

## 4. Встановити залежності

```bash
pip install -r requirements.txt
```

---

## 5. Запустити Redis

```bash
redis-server
```

---

## 6. Запустити проєкт

```bash
python run.py
```

або

```bash
uvicorn app.main:app --reload
```

# 🔐 Авторизація

Авторизація реалізована за допомогою JWT Token.

Після входу токен автоматично використовується для доступу до захищених маршрутів.

---

# 💳 Бронювання

1. Користувач обирає місця.
2. Місця резервуються у Redis на 10 хвилин.
3. Після оплати створюються квитки.
4. Генерується QR-код.
5. Квитки зберігаються в особистому кабінеті.
